Load code into Arduino
Add libraries
Compile and load into Arduino Nano connected to 1.8" ST7735
Encoder and 3 buttons to be used in this project
Connections as follows:
tft ArdNano
LED     3.3v
SCK     D13
SDA     D11
A0      D8
RST     D9
CS      D10
GND     GND
VCC     5V

Encoder ArdNano
CW      D2
CCW     D3

Parameters buttons 1, 2, 3, other button's pins connected to GND
Button 1  - A0
Button 2  - A1
Button 3  - A2

PWM:
POT1 - D5
POT2 - D6
POT3 - D12

Outputs:
S0   - A4
S1   - A5
S2   - A6
Rom1 - A7
Rom2 - A8
